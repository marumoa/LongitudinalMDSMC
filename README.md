# LongitudinalMDSMC
We used machine learning to classify MDS cases into molecular clusters based on genetic information and evaluated their longitudinal changes. We provide the scripts required to generate the figures included in the manuscript.

#Notes
This study evaluated longitudinal changes in molecular clusters based on the molecular classification previously reported by our group in Nature Communications (Nat Commun. 2023;14:3136. doi:10.1038/s41467-023-38515-4). Therefore, the method for molecular cluster classification is provided separately at https://github.com/ardadurmaz/mds_latent
The data used in the present study have been uploaded to https://doi.org/10.6084/m9.figshare.32117959

We provide the RStudio scripts used to generate Figure 2A–C, Figure 3B, and Supplementary Figure S4.
